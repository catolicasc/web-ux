Política de Privacidade

Extensão: Sorteador de Grupos para Turmas

Última atualização: 02 de março de 2026

Resumo

  • Não coletamos dados pessoais.
  • Não enviamos dados para servidores externos.
  • Acesso apenas à aba ativa quando você clica na extensão.
  • Preferências ficam no seu navegador (storage local).

1. Introdução

A extensão Sorteador de Grupos para Turmas foi desenvolvida com o objetivo exclusivo de auxiliar professores a criar grupos de alunos de forma automática e aleatória diretamente na página da turma, reduzindo trabalho manual.

2. Coleta de Informações

Esta extensão não coleta, não vende e não compartilha dados pessoais. A extensão apenas lê temporariamente as informações que já estão visíveis na página ativa no momento em que o usuário executa a ação de sorteio.

Em resumo:

  • Não coletamos nomes para nossos servidores
  • Não registramos histórico de navegação
  • Não usamos cookies
  • Não fazemos rastreamento

3. Como as Informações são Usadas

Quando você clica para sortear, a extensão lê a lista de alunos exibida na página e utiliza esses dados somente para:

  • Aplicar filtros (ex.: considerar apenas alunos matriculados)
  • Fixar líderes pré-definidos em seus respectivos grupos
  • Distribuir os alunos aleatoriamente entre os grupos
  • Exibir e permitir copiar o resultado formatado

O processamento ocorre localmente no seu navegador.

4. Armazenamento Local

A extensão pode utilizar armazenamento local do navegador (chrome.storage) para salvar preferências, como:

  • Número de grupos configurado
  • Líderes fixos definidos pelo usuário
  • Preferência de filtros

Essas informações ficam apenas no seu navegador e não são sincronizadas com servidores externos pela extensão.

5. Compartilhamento com Terceiros

A extensão não compartilha dados com terceiros e não integra ferramentas de analytics, publicidade, rastreamento ou coleta de telemetria.

6. Permissões Utilizadas

A extensão utiliza permissões mínimas necessárias para funcionar:

activeTab

Permite acesso temporário à aba ativa apenas quando você clica na extensão, para ler a lista exibida na página e gerar os grupos.

scripting

Permite executar um script na página ativa para extrair os nomes no DOM e montar a distribuição dos grupos conforme suas configurações.

storage

Usada para salvar preferências de uso localmente (ex.: número de grupos, líderes fixos e filtros), evitando reconfigurar a cada sorteio.

Permissão de Host (site alvo)

Necessária para permitir que a extensão funcione no(s) domínio(s) onde a lista de alunos é exibida, limitando o acesso apenas às páginas necessárias para o sorteio.

7. Segurança

O processamento ocorre localmente e a extensão não envia dados pela rede. Ainda assim, recomendamos manter o navegador atualizado e instalar extensões apenas de fontes confiáveis.

8. Alterações nesta Política

Podemos atualizar esta Política de Privacidade para refletir melhorias na extensão ou exigências de plataforma. A data de “Última atualização” no topo indicará a versão mais recente.
